# Data Collection Guide

## Quick Start

1. Prepare CSV file with 21 fields
2. Collect TikTok data (10 products)
3. Upload via Dashboard sidebar
4. Click 'Save to Google Drive'

## CSV Format
File name: All_Data_Week_XX.csv
Fields: week_number, year, report_date, product_rank, product_category, product_name, total_score, views_score, engagement_score, trend_score, demand_score, views, likes, comments, shares, engagement_rate, tiktok_url, ai_market_positioning, ai_target_audience, ai_pricing_strategy, ai_risks

## Storage
All data saved to: Google Drive/3d-printing-data/week_XX/

See data_structure_analysis.md for details.
